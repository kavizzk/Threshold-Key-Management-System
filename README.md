# 🔐 AI-Based Threshold Key Management System (TKMS)

## 🎓 Final Year Project

**AI-Based Threshold Key Management System (TKMS)** is a Final Year B.Tech project developed to provide a secure, distributed, and intelligent approach to cryptographic key management in cloud environments.

The system combines **Threshold Cryptography**, **Shamir's Secret Sharing**, **Artificial Intelligence-based Trust Evaluation**, and **AES Encryption** to improve security, fault tolerance, and reliability while eliminating the single point of failure found in traditional key management systems.

---

## 📌 Project Overview

Traditional Key Management Systems (KMS) store encryption keys centrally, making them vulnerable to attacks and system failures.

This project addresses these challenges by:
- Splitting the secret key into multiple shares using **Shamir's Secret Sharing**
- Distributing key shares across multiple **Key Management Nodes (KMNs)**
- Monitoring node behavior using an **AI-based Trust Evaluation** mechanism
- Reconstructing the key only from trusted nodes
- Encrypting and decrypting data securely using **AES**
